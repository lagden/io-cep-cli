## CLI

![Demo cep CLI](https://raw.githubusercontent.com/lagden/io-cep-cli/master/demo.gif)


#### Install

```
$ npm i -g io-cep-cli
```


#### Usage

```
$ cep 04080001
```

or

```
$ cep 04080-001
```
